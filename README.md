# MiWebMSFS20

Proyecto personal de web VFR para Microsoft Flight Simulator 2020.

- Enfoque: VFR visual, disfrute y aprendizaje
- Avión base: Cessna 152
- Planificación externa: Little Navmap
- Web estática (HTML/CSS/JS)
- Publicada en GitHub Pages
- SimConnect previsto a futuro mediante programa puente (no ahora)
